# AgentFlow - 智能体工作流平台

一个功能完整的智能体工作流可视化编排平台,基于 Vue3 + LogicFlow 构建。

## ✨ 核心特性

### 🎨 可视化流程编排
- 拖拽式流程编辑器,支持多种节点类型
- 开始/结束节点、Agent执行节点、条件分支、子流程、并行网关
- 灵活的连线配置和条件表达式编辑
- 直观的画布操作:缩放、平移、适应视图

### 🤖 多Agent协同
- Agent节点参数化配置面板
- 支持多个Agent串行/并行编排
- Agent能力注册与管理机制
- 支持自定义Agent类型和配置

### 🔀 条件分支处理
- 可视化条件表达式编辑器
- 运行时动态路径决策
- 支持多分支条件判断

### 📦 子流程管理
- 独立的子流程创建和编辑
- 支持嵌套调用和参数传递
- 输入输出映射配置

### 🐛 强大的调试功能
- 实时执行日志追踪
- 单步执行与断点调试
- 可视化执行状态指示
- 详细的错误诊断信息

## 🏗️ 技术架构

### 技术栈
- **前端框架**: Vue 3 (Composition API + setup语法糖)
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **流程引擎**: LogicFlow 2.0+
- **代码规范**: ESLint + Prettier

## 📖 使用指南

### 1. 创建工作流
- 启动应用后会自动创建一个新的工作流
- 可以在顶部工具栏修改工作流名称

### 2. 添加节点
- 从左侧节点面板拖拽节点到画布
- 支持的节点类型:
  - **开始节点**: 工作流起点
  - **结束节点**: 工作流终点
  - **Agent节点**: 执行智能体任务
  - **条件分支**: 根据条件选择路径
  - **子流程**: 调用其他工作流
  - **并行网关**: 并行执行多个分支

### 3. 配置节点
- 点击节点查看右侧属性面板
- 根据节点类型配置相应参数:
  - Agent节点: 名称、类型、提示词、温度等
  - 条件节点: 条件表达式和分支配置
  - 子流程: 选择要调用的工作流

### 4. 连接节点
- 拖拽节点的连接点创建连线
- 条件节点可以配置不同分支的条件

### 5. 运行工作流
- 点击顶部工具栏的"运行"按钮
- 查看底部执行日志面板的实时输出
- 支持查看每个节点的执行结果

### 6. 调试模式
- 切换到"调试"模式
- 在属性面板为节点添加断点
- 使用调试控制栏进行单步执行、继续等操作

### 7. 保存与导出
- 点击"保存"按钮保存当前工作流
- 点击"导出"按钮导出为JSON文件
- 工作流自动保存到浏览器本地存储

## 🔧 核心功能实现

### 状态管理
使用 Pinia 实现两个核心状态仓库:
- **workflowStore**: 管理工作流数据、执行状态、日志等
- **nodeStore**: 管理节点类型、已注册Agent、选中状态等

严格区分编辑态和运行态,确保状态隔离。

### 工作流引擎
自研的工作流执行引擎,支持:
- 顺序执行
- 条件分支
- 并行执行
- 子流程调用
- 断点调试
- 单步执行

### LogicFlow集成
- 自定义节点样式和行为
- 注册不同类型的节点组件
- 监听画布事件进行业务逻辑处理
- 实现节点高亮和执行状态展示

## 📝 开发规范

### 代码风格
- 使用 Vue 3 Composition API
- 采用 setup 语法糖
- 遵循 ESLint + Prettier 规范
- 组件名使用 PascalCase
- 文件名使用 camelCase 或 PascalCase

### 组件设计原则
- 高内聚低耦合
- 单一职责
- 基础组件与业务组件分离
- Props 类型定义清晰
- 事件命名语义化
